Helper Methods

static void Initialize (string tfsEndpoint, string tfsTeamProjectName, string rootAreaPath, string tfsDomain, string tfsUser, string tfsPassword)
  Initializes the ODataForTFS helper with the parameters for the OData Service for TFS and the Team Foundation Server.
@helper GetHeaderHtml ([string helperCssSheet])
  Outputs the necessary HTML headers for using the ODataForTFS helper. The path of the CSS sheet for the helper can be specified to customize the defualt UI.
@helper GetKnownIssuesHtml (string contentKey, string areaPath, [string getPage])
  Displays the Known Issues icon on the page. When the user clicks it, shows a modal dialog with all the bugs for the current context page.
@helper GetFileBugHtml (string contentKey, string areaPath, [string savePage], [string areasPage])
  Displays the File Bug icon on the page. When the user clicks it, shows a modal dialog to allow the user to submit basic information about the new bug.
static void AddWorkItem (string title, string areaPath, string priority, string severity, string reproOrDescription, [string type])
  Creates a new work item and stores it using the OData Service for TFS.
static IList<dynamic> GetWorkItems (string areaPath, [int pageNumber = 0], [string workItemType])
  Returns a list of the work items in TFS for the areaPaths specified. If a pageNumber is provided, returns only the work items for that page, otherwise returns the first page.
static IList<dynamic> GetAreaPaths ()
  Returns a list of the available TFS area paths for the TFS Team Project configured.
static void SaveContentFromPost ()
  Stores content data sent from a form POST.
